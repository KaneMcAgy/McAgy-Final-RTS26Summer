# McAgy-Final-RTS26Summer

Final Capstone Repository for EEL 4775 Real-Time Systems.

## Project Overview

This repository contains the AEGIS Avionics Flight Control System, a FreeRTOS-based embedded application developed on the ESP32-S3 using Wokwi. The project demonstrates periodic task scheduling, interrupt handling, inter-task communication, and runtime performance measurement.
